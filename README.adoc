= zaster2

Collection of tools to import transaction data from crypto exchanges or blockchains

== Setup database

----
sudo mysql -u root
CREATE DATABASE zaster2;
SHOW DATABASES;

GRANT ALL PRIVILEGES ON zaster2.* TO 'bot'@'localhost' WITH GRANT OPTION;
FLUSH PRIVILEGES;
SHOW GRANTS FOR bot@localhost;
----

== Export database

----
mysqldump -u bot -p --single-transaction --quick --routines --triggers --events zaster2 | gzip -c > zaster2.sql.gz
----

== Import database

----
gunzip -c zaster2.sql.gz | mysql -u bot -p zaster2
----
